Application Fitness

Une application de suivi sportif pour gérer les séances, les exercices, la progression et l’affichage clair des données.

Application Fitness
Problème

Le suivi sportif peut vite devenir dispersé entre notes, applications et tableaux. Le besoin était de centraliser les séances et la progression dans une interface simple.

Solution

Conception d’un modèle de données clair avec CRUD, pages lisibles, validation et logique métier autour des séances, exercices et statistiques.

Impact

Le projet démontre une capacité à structurer des données, créer une logique métier et construire une base prête pour de futures statistiques ou analytics.

Mon rôle

Conception des entités, structuration SQL/MySQL, développement CRUD, validation des données et interface utilisateur.
