Jeu Snake

Un Snake fluide avec score, collisions, difficulté progressive et gameplay propre en HTML, CSS et JavaScript.

JavaScript HTML CSS Game loop Front-end
Jeu Snake
Problème

Le défi était de créer un mini-jeu stable, fluide et réactif sans framework lourd, avec une logique claire et une gestion propre des états.

Solution

Mise en place d’une boucle de jeu maîtrisée, gestion des contrôles clavier, règles de collision, score et états de fin de partie.

Impact

Le projet montre la maîtrise de la logique front-end, de la gestion d’état, de l’interaction utilisateur et de la structuration d’un code JavaScript simple mais propre.

Mon rôle

Développement JavaScript, logique de jeu, interface, scoring, gestion des collisions et amélioration progressive du gameplay.
